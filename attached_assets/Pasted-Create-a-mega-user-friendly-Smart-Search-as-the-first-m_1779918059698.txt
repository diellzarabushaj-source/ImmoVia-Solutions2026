Create a mega user-friendly Smart Search as the first main element on the Home page.

This is a FRONTEND/UI + SEARCH ROUTING task.

IMPORTANT:
Frontend must be very user-friendly.
Backend must stay strict.

Do not change the database.
Do not rename variables.
Do not create new roles.
Do not change authentication.
Do not change admin logic.
Do not change account_type, account_subtype, role, or status.

Backend values must remain:

account_type = project_poster/service_provider
