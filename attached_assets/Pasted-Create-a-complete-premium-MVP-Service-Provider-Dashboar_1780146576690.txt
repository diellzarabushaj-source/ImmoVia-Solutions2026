Create a complete premium MVP Service Provider Dashboard and Public Provider Profile system for ImmoVia365.

ImmoVia365 is a Swiss marketplace that connects project owners with handymen, renovation companies, construction companies and service providers in Switzerland.

The goal is to make every Service Provider profile look professional, trustworthy and useful, almost like a mini business website inside ImmoVia365.

This system is only for users with the role “Service Provider / Anbieter”.

Customers / Project Posters must have a separate dashboard and must not see provider pricing, billing, upgrade options or provider subscription management.

DEFAULT LANGUAGE

The default platform language is German.

The platform should also support:

* Deutsch
* English
* Shqip
* Français

All visible frontend text must be translatable.

Use German as the default language for all labels, buttons and messages.

Design style:

* Clean Swiss marketplace look
* White or very light background
* Dark navy typography
* Bright blue accent color
* Rounded cards
* Professional profile layout
* Clear CTAs
* Mobile responsive
* Trustworthy and premium
* Simple enough for MVP
